## Usage

1. Open the Standard Notes Desktop App or visit <a href="https://app.standardnotes.org/">Standard Notes Web</a>.
2. Click the Quick Settings button in the lower left, and select "Open Preferences".
3. Click General and scroll down to select Advanced Options.
4. In the "Install External Package" field, paste this URL: `https://raw.githubusercontent.com/rio-codes/catppuccin-standardnotes/main/ext.json` and click Install.
5. Close Preferences, and in the Quick Settings menu select Themes -> Catppuccin.
